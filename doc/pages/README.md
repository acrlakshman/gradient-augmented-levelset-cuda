## Installation

```sh
   git clone https://github.com/acrlakshman/gradient-augmented-levelset-cuda --recursive
   cd gradient-augmented-levelset-cuda
   mkdir -p build && cd build
   cmake .. -GNinja && ninja -j 4
```
